## 🎮 Simon Game

An interactive memory game inspired by the classic Simon game, built using HTML, CSS, and JavaScript.

The game challenges players to remember and repeat an increasingly complex sequence of colors and sounds. With each level, the difficulty increases, testing the player’s memory and focus.

### 🚀 Features
- Interactive gameplay with sound and visual effects
- Increasing difficulty with each level
- Smooth and responsive UI
- Simple and engaging user experience

### 🎯 How to Play
1. Press any key to start the game
2. Watch the sequence of colors
3. Repeat the sequence by clicking the buttons
4. Each correct sequence advances you to the next level
5. A wrong move ends the game

### ⚠️ Note
This project is built practice purposes, focusing on JavaScript logic and DOM manipulation.
